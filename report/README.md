# `report/`

Optional copies of correlation run outputs, same general shape as [`output/correlate/…/`](../output/README.md) (e.g. `…/report.json`, per-metric `rolling.csv` and PNGs under metric folders).

Not generated by CI in this repository; content is for local or manual archiving. The canonical path description and artifact list live in [output/README.md](../output/README.md).

See [AGENTS.md](AGENTS.md) for folder-level notes.
